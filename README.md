# Проект по парсингу статического сайта

## Описание
Автоматический парсер статического веб-сайта, загружающий собранную информацию в Google Таблицы.

## Структура проекта

```
static-price-parser/
├── parser.py          # Основной скрипт парсинга веб-сайта
├── task.py            # Скрипт-обёртка для автоматизации запуска
├── task.txt           # Лог-файл с временем запуска парсера
├── secrets/
│   └── cred.json      # JSON-файл с учетными данными Google Sheets API
├── venv/              # Виртуальное окружение Python
└── requirements.txt   # Список зависимостей
```

## Способ работы

**parser.py** - программа по парсингу веб-сайта, извлекает название товара, цену и наличие, после чего записывает полученную информацию в Google Таблицы и добавляет время выгрузки.

**task.py** - скрипт для автоматизации запуска парсера при помощи Планировщика задач Windows.

**task.txt** - лог-файл с временем начала выполнения каждого запуска парсера.
